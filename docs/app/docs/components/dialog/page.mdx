# Dialog 对话框

对话框组件用于在当前页面上方显示一个模态窗口，常用于确认操作、表单填写等场景。

## 安装

<InstallDependencies packages={['@radix-ui/react-dialog']} />

## 导入

```ts copy
import {

  DialogTrigger,

  DialogHeader,
  DialogTitle,
  DialogDescription,
  DialogFooter,
  DialogClose,
```

## API

### Dialog Props

简化的对话框组件。

| 属性 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| trigger | `ReactNode` | - | 触发对话框的元素 |
| title | `ReactNode` | - | 对话框标题 |
| description | `ReactNode` | - | 对话框描述 |
| footer | `ReactNode` | - | 底部内容 |
| open | `boolean` | - | 是否打开（受控） |
| defaultOpen | `boolean` | `false` | 默认是否打开 |
| onOpenChange | `(open: boolean) => void` | - | 状态变化回调 |
| classNames | `DialogClassNames` | - | 自定义各部分的样式类名 |
| className | `ClassValue` | - | 根容器自定义类名 |
| contentProps | `DialogContentProps` | - | 内容区域属性 |

> 支持 Radix UI Dialog 的其他所有属性。

### DialogClassNames

| 属性 | 类型 | 说明 |
|------|------|------|
| overlay | `ClassValue` | 遮罩层样式 |
| content | `ClassValue` | 内容区域样式 |
| header | `ClassValue` | 头部样式 |
| title | `ClassValue` | 标题样式 |
| description | `ClassValue` | 描述样式 |
| footer | `ClassValue` | 底部样式 |
| close | `ClassValue` | 关闭按钮样式 |

### DialogContent Props

| 属性 | 类型 | 说明 |
|------|------|------|
| className | `ClassValue` | 自定义类名 |
| onOpenAutoFocus | `(e: Event) => void` | 打开时自动聚焦回调 |
| onCloseAutoFocus | `(e: Event) => void` | 关闭时自动聚焦回调 |
| onEscapeKeyDown | `(e: KeyboardEvent) => void` | ESC 键按下回调 |
| onPointerDownOutside | `(e: PointerEvent) => void` | 点击外部回调 |
| onInteractOutside | `(e: Event) => void` | 外部交互回调 |

## 使用指南

### 何时使用

- 确认危险操作
- 表单填写
- 详情查看
- 设置面板

## 无障碍

- 自动焦点陷阱
- 支持 ESC 关闭
- ARIA 属性完善
- 背景滚动锁定

## 最佳实践

### 确认对话框

```tsx
<Dialog
  title="确认删除"
  description="此操作不可撤销，确定要删除吗？"
  trigger={<Button variant="destructive">删除</Button>}
  footer={(
    <>
      <DialogClose asChild>
        <Button variant="plain">取消</Button>
      </DialogClose>
      <Button variant="destructive" onClick={handleDelete}>
        确认删除
      </Button>
    </>
  )}
/>
```

### 表单对话框

```tsx
<Dialog
  title="编辑资料"
  trigger={<Button>编辑</Button>}
  footer={(
    <>
      <DialogClose asChild>
        <Button variant="plain">取消</Button>
      </DialogClose>
      <Button type="submit" form="edit-form">
        保存
      </Button>
    </>
  )}
>
  <form id="edit-form" onSubmit={handleSubmit}>
    <Input name="name" placeholder="姓名" />
    <Input name="email" placeholder="邮箱" />
  </form>
</Dialog>
```

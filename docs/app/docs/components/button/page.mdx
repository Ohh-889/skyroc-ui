# Button 按钮

按钮用于开始一个即时操作。

## 导入

```tsx
import { Button, ButtonGroup, ButtonIcon } from 'skyroc-ui';
```

## API

### Button Props

| 属性 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| color | `'primary' \| 'destructive' \| 'success' \| 'warning' \| 'info' \| 'carbon' \| 'secondary' \| 'accent'` | `'primary'` | 按钮颜色 |
| variant | `'solid' \| 'pure' \| 'plain' \| 'outline' \| 'dashed' \| 'soft' \| 'ghost' \| 'link'` | `'solid'` | 按钮变体 |
| size | `'xs' \| 'sm' \| 'md' \| 'lg' \| 'xl' \| '2xl'` | `'md'` | 按钮尺寸 |
| shape | `'auto' \| 'rounded' \| 'square' \| 'circle'` | `'auto'` | 按钮形状 |
| shadow | `'none' \| 'sm' \| 'md' \| 'lg'` | `'sm'` | 阴影大小 |
| fitContent | `boolean` | `false` | 是否自适应内容 |
| loading | `boolean` | `false` | 加载状态 |
| disabled | `boolean` | `false` | 禁用状态 |
| leading | `ReactNode` | - | 前置内容（图标等） |
| trailing | `ReactNode` | - | 后置内容（图标等） |
| asChild | `boolean` | `false` | 作为子元素渲染（使用 Radix UI Slot） |

### ButtonGroup Props

| 属性 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| orientation | `'horizontal' \| 'vertical'` | `'horizontal'` | 按钮组方向 |

### ButtonIcon Props

继承 `Button` 的所有属性，并有以下特殊默认值：

| 属性 | 默认值 | 说明 |
|------|--------|------|
| color | `'accent'` | 图标按钮默认颜色 |
| variant | `'ghost'` | 图标按钮默认变体 |
| shape | `'square'` | 图标按钮默认形状 |
| fitContent | `true` | 图标按钮默认自适应内容 |

额外属性：

| 属性 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| icon | `string` | - | 图标名称（使用 Icon 组件） |
| iconProps | `IconProps` | - | 传递给 Icon 组件的属性 |

## 变体说明

### solid（实心）
默认样式，有背景色和文字颜色对比。

### pure（纯净）
有边框和背景，适合表单场景。

### plain（朴素）
有边框无背景，鼠标悬浮时边框和文字改变颜色。

### outline（描边）
有边框和颜色，背景透明。

### dashed（虚线）
虚线边框，适合次要操作。

### soft（柔和）
浅色背景，无边框。

### ghost（幽灵）
完全透明背景，鼠标悬浮时显示背景。

### link（链接）
类似文本链接，带下划线效果。

## 颜色系统

按钮支持主题色系统：

- **primary** - 主要操作
- **destructive** - 危险/删除操作
- **success** - 成功/确认操作
- **warning** - 警告操作
- **info** - 信息提示
- **carbon** - 深色主题
- **secondary** - 次要操作
- **accent** - 强调色

## 最佳实践

### 使用场景

- **solid** - 主要操作按钮
- **outline** - 次要操作按钮
- **ghost** - 工具栏按钮
- **link** - 内联操作

### 颜色选择

- **primary** - 主要操作（提交、保存）
- **destructive** - 删除、取消订阅等危险操作
- **success** - 确认、完成
- **warning** - 需要谨慎的操作

### 尺寸选择

- **xs/sm** - 表格行内操作、标签
- **md** - 默认尺寸，适合大多数场景
- **lg/xl** - 重要的号召性操作（CTA）
- **2xl** - 大型页面的主要操作

## 注意事项

1. 加载状态（`loading`）会自动禁用按钮
2. `fitContent` 设为 `true` 时，按钮会自适应内容大小
3. `shape` 为 `square` 或 `circle` 时，建议只放图标
4. 使用 `asChild` 可以将按钮样式应用到其他组件上

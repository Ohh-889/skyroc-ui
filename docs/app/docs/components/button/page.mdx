# Button 按钮

按钮用于触发即时操作，是用户界面中最常用的交互元素。

## 安装

<InstallDependencies packages={['@radix-ui/react-slot', 'lucide-react']} />

## 导入

```ts copy
import { Button, ButtonGroup, ButtonIcon } from 'skyroc-ui';
```

## 基础用法

### 颜色

按钮提供 8 种主题颜色，每种颜色都有其特定的语义和使用场景。

<Demo src="@playground/button/modules/ButtonColor" title="按钮颜色" />

### 变体

按钮提供 8 种视觉变体样式，可以满足不同的设计需求和视觉层级。

<Demo src="@playground/button/modules/ButtonVariant" title="按钮变体" />

### 尺寸

按钮提供 6 种尺寸大小：`xs`、`sm`、`md`、`lg`、`xl`、`2xl`。

<Demo src="@playground/button/modules/ButtonSize" title="按钮尺寸" />

### 形状

按钮提供多种形状：`auto`（默认）、`rounded`（完全圆角）、`square`（正方形）、`circle`（圆形）。

<Demo src="@playground/button/modules/ButtonShape" title="按钮形状" />

### 阴影

按钮提供 4 种阴影层级：`none`、`sm`、`md`、`lg`，用于增强按钮的视觉层次。

<Demo src="@playground/button/modules/ButtonShadow" title="按钮阴影" />

### 前后插槽

按钮支持 `leading` 和 `trailing` 插槽，可以在按钮前后添加图标或其他内容。

<Demo src="@playground/button/modules/ButtonSlot" title="按钮插槽" />

### 禁用状态

通过 `disabled` 属性禁用按钮，禁用后按钮不可点击且样式变灰。

<Demo src="@playground/button/modules/ButtonDisabled" title="禁用按钮" />

### 加载状态

通过 `loading` 属性显示加载状态，加载时按钮会自动禁用并显示加载图标。

<Demo src="@playground/button/modules/ButtonLoading" title="加载按钮" />

### 内容自适应

通过 `fitContent` 属性控制按钮是否自适应内容大小，适合只包含图标的场景。

```tsx
<Button fitContent>
  <Icon icon="heart" />
</Button>
```

### 作为其他组件

通过 `asChild` 属性，可以将按钮样式应用到其他组件上，如链接。

```tsx
<Button asChild>
  <a href="/docs">文档</a>
</Button>
```

## API

### Button Props

| 属性 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| color | `'primary' \| 'destructive' \| 'success' \| 'warning' \| 'info' \| 'carbon' \| 'secondary' \| 'accent'` | `'primary'` | 按钮颜色主题 |
| variant | `'solid' \| 'pure' \| 'plain' \| 'outline' \| 'dashed' \| 'soft' \| 'ghost' \| 'link'` | `'solid'` | 按钮变体样式 |
| size | `'xs' \| 'sm' \| 'md' \| 'lg' \| 'xl' \| '2xl'` | `'md'` | 按钮尺寸大小 |
| shape | `'auto' \| 'rounded' \| 'square' \| 'circle'` | `'auto'` | 按钮形状 |
| shadow | `'none' \| 'sm' \| 'md' \| 'lg'` | `'sm'` | 阴影层级 |
| fitContent | `boolean` | `false` | 是否自适应内容大小 |
| loading | `boolean` | `false` | 加载状态 |
| disabled | `boolean` | `false` | 禁用状态 |
| leading | `ReactNode` | - | 前置内容（图标等） |
| trailing | `ReactNode` | - | 后置内容（图标等） |
| asChild | `boolean` | `false` | 作为子元素渲染（使用 Radix UI Slot） |

### ButtonGroup Props

| 属性 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| orientation | `'horizontal' \| 'vertical'` | `'horizontal'` | 按钮组排列方向 |

### ButtonIcon Props

继承 `Button` 的所有属性，并有以下特殊默认值：

| 属性 | 默认值 | 说明 |
|------|--------|------|
| color | `'accent'` | 图标按钮默认颜色 |
| variant | `'ghost'` | 图标按钮默认变体 |
| shape | `'square'` | 图标按钮默认形状 |
| fitContent | `true` | 图标按钮默认自适应内容 |

额外属性：

| 属性 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| icon | `string` | - | 图标名称（使用 Icon 组件） |
| iconProps | `IconProps` | - | 传递给 Icon 组件的属性 |

## 使用指南

### 何时使用

- 需要触发即时操作时，如提交表单、打开对话框等
- 需要明确的用户交互入口
- 需要在界面中突出某个操作

### 变体选择建议

- **Solid** - 主要操作按钮，最高优先级，一个页面最好只有一个
- **Outline** - 次要操作按钮，与 Solid 搭配使用
- **Ghost** - 工具栏按钮、图标按钮，最低视觉权重
- **Link** - 内联操作，不需要明显按钮样式的场景
- **Dashed** - 添加、上传等非永久性操作
- **Soft** - 需要柔和视觉效果的场景

### 颜色使用建议

- **Primary** - 主要操作（提交、保存、确认）
- **Destructive** - 危险操作（删除、取消、拒绝）
- **Success** - 成功、完成类操作
- **Warning** - 需要谨慎的操作
- **Info** - 信息类操作
- **Secondary** - 次要操作
- **Accent** - 需要强调但不属于主要操作的场景

### 尺寸使用建议

- **xs/sm** - 表格行内操作、标签、紧凑型界面
- **md** - 默认尺寸，适合大多数场景
- **lg/xl** - 重要的号召性操作（CTA）、表单提交
- **2xl** - 大型页面的主要操作、营销页面

## 注意事项

1. **加载状态** - `loading` 为 `true` 时会自动禁用按钮并显示加载图标
2. **禁用状态** - 禁用的按钮不可点击，且 `opacity` 降低为 50%
3. **焦点管理** - 所有按钮都支持键盘焦点和焦点环（focus ring）
4. **形状与内容** - `square` 或 `circle` 形状时，建议只放图标
5. **自适应内容** - `fitContent` 为 `true` 时，按钮会根据内容大小自适应
6. **语义化** - 使用 `asChild` 时要注意保持语义化，如链接应该用 `<a>` 标签

## 无障碍

- 按钮支持键盘访问（Space/Enter 键触发）
- 禁用的按钮会添加 `disabled` 属性
- 加载状态的按钮会自动禁用，防止重复提交
- 所有按钮都有明确的焦点指示器
- 建议为图标按钮添加 `aria-label` 属性以提供文本描述

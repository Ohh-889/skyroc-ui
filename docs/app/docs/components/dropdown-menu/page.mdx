# DropdownMenu 下拉菜单

下拉菜单组件用于在触发元素下方显示操作选项列表。

## 安装

<InstallDependencies packages={['@radix-ui/react-dropdown-menu']} />

## 导入

```ts copy
import { 
  DropdownMenu,
  DropdownMenuRoot,
  DropdownMenuTrigger,
  DropdownMenuContent,
  DropdownMenuItem,
  DropdownMenuSeparator,
  DropdownMenuCheckbox,
  DropdownMenuRadio,
  DropdownMenuLabel,
  DropdownMenuArrow
} from 'skyroc-ui';
```

## API

### DropdownMenu Props

简化的下拉菜单组件。

| 属性 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| items | `DropdownMenuOptionProps['item'][]` | - | 菜单项数据 |
| contentProps | `DropdownMenuContentProps` | - | 菜单内容区域属性 |
| open | `boolean` | - | 是否打开（受控） |
| defaultOpen | `boolean` | - | 默认是否打开 |
| onOpenChange | `(open: boolean) => void` | - | 状态变化回调 |
| modal | `boolean` | `true` | 是否为模态 |
| className | `ClassValue` | - | 自定义类名 |

> 支持 Radix UI DropdownMenu 的其他所有属性。

### 菜单项类型

菜单项支持多种类型：

```ts
// 普通菜单项
{ label: '编辑', shortcut: '⌘E', onSelect: () => {} }

// 分隔线
{ type: 'separator' }

// 子菜单
{ label: '更多', children: [...] }

// 禁用项
{ label: '删除', disabled: true }
```

### DropdownMenuItem Props

| 属性 | 类型 | 说明 |
|------|------|------|
| label | `ReactNode` | 菜单项标签 |
| shortcut | `string \| string[]` | 快捷键提示 |
| disabled | `boolean` | 是否禁用 |
| onSelect | `() => void` | 点击回调 |
| className | `ClassValue` | 自定义类名 |

### DropdownMenuCheckbox Props

复选菜单组件。

| 属性 | 类型 | 说明 |
|------|------|------|
| checked | `boolean` | 是否选中 |
| onCheckedChange | `(checked: boolean) => void` | 状态变化回调 |
| items | `CheckboxItemProps[]` | 复选项列表 |

### DropdownMenuRadio Props

单选菜单组件。

| 属性 | 类型 | 说明 |
|------|------|------|
| value | `string` | 当前选中值 |
| onValueChange | `(value: string) => void` | 选中值变化回调 |
| items | `RadioItemProps[]` | 单选项列表 |

## 使用指南

### 何时使用

- 操作按钮的扩展菜单
- 用户头像下拉菜单
- 更多操作入口
- 设置选项

## 无障碍

- 支持键盘导航（上下箭头）
- 支持 ESC 关闭
- 支持 Enter/Space 选择
- ARIA 属性完善

## 最佳实践

### 用户菜单

```tsx
const items = [
  { label: '个人资料', shortcut: '⌘P' },
  { label: '设置', shortcut: '⌘,' },
  { type: 'separator' },
  { label: '退出登录', shortcut: '⌘Q' }
];

<DropdownMenu items={items}>
  <Avatar src={user.avatar} />
</DropdownMenu>
```

### 带子菜单

```tsx
const items = [
  { label: '新建', children: [
    { label: '文件' },
    { label: '文件夹' }
  ]},
  { type: 'separator' },
  { label: '设置' }
];

<DropdownMenu items={items}>
  <Button>操作</Button>
</DropdownMenu>
```

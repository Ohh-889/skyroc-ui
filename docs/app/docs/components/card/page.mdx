# Card 卡片

卡片是一个容器组件，用于将相关内容和操作组织在一起，常用于展示信息摘要、产品列表等场景。

## 安装

Card 组件是内置组件，无需额外安装依赖。

## 导入

```ts copy
import { 
  Card,
  CardRoot,
  CardHeader,
  CardTitleRoot,
  CardTitle,
  CardContent,
  CardFooter
} from 'skyroc-ui';
```

## 组合式 API

如果需要更灵活的布局控制，可以使用组合式 API：

```tsx
import {
  CardRoot,
  CardHeader,
  CardTitleRoot,
  CardTitle,
  CardContent,
  CardFooter
} from 'skyroc-ui';

export default function Demo() {
  return (
    <CardRoot>
      <CardHeader>
        <CardTitleRoot>
          <CardTitle>卡片标题</CardTitle>
        </CardTitleRoot>
      </CardHeader>
      <CardContent>
        <p>卡片内容</p>
      </CardContent>
      <CardFooter>
        <p>卡片底部</p>
      </CardFooter>
    </CardRoot>
  );
}
```

## API

### Card Props

简化的卡片组件，封装了所有子组件。

| 属性 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| title | `ReactNode` | - | 卡片标题 |
| titleLeading | `ReactNode` | - | 标题前置内容（如图标） |
| titleTrailing | `ReactNode` | - | 标题后置内容（如徽章） |
| titleRoot | `ReactNode` | - | 完全自定义标题区域 |
| header | `ReactNode` | - | 完全自定义头部区域 |
| footer | `ReactNode` | - | 底部内容 |
| extra | `ReactNode` | - | 头部右侧额外内容 |
| size | `'xs' \| 'sm' \| 'md' \| 'lg' \| 'xl' \| '2xl'` | `'md'` | 卡片尺寸 |
| split | `boolean` | `false` | 是否显示分割线 |
| flexHeight | `boolean` | `false` | 内容区域是否弹性高度 |
| classNames | `CardUi` | - | 自定义各部分的样式类名 |
| className | `ClassValue` | - | 根容器自定义类名 |

> 支持原生 `div` 元素的其他所有属性。

### CardUi（classNames）

| 属性 | 类型 | 说明 |

## 使用指南

### 何时使用

- 展示信息摘要或预览
- 产品列表、文章列表
- 表单容器、设置面板
- 仪表盘小部件

### 尺寸使用建议

- **xs/sm** - 紧凑型布局、列表项
- **md** - 默认尺寸，适合大多数场景
- **lg/xl** - 重要内容、详情展示
- **2xl** - 大型展示、营销页面

### 分割线使用建议

- 当头部、内容、底部需要明确区分时使用 `split`
- 简单卡片可以不使用分割线，保持简洁

## 注意事项

1. **标题简洁** - 卡片标题应该简短明了
2. **内容适量** - 卡片内容不宜过多，保持信息密度适中
3. **操作明确** - 底部或 extra 区域的操作按钮应该清晰可见
4. **弹性高度** - 使用 `flexHeight` 时确保父容器有固定高度

## 无障碍

- 卡片使用 `div` 元素，无特殊 ARIA 属性
- 确保卡片内的链接和按钮可以通过键盘访问
- 如果卡片是可点击的，应该使用 `button` 或 `a` 元素包裹

## 最佳实践

### 信息卡片

```tsx
<Card
  split
  title="项目信息"
  extra={<Button variant="ghost" size="sm">编辑</Button>}
  footer={<span className="text-muted-foreground text-sm">最后更新: 2024-01-01</span>}
>
  <p>项目描述内容...</p>
</Card>
```

### 产品卡片

```tsx
<Card split>
  <img src="/product.jpg" alt="产品图片" className="w-full h-48 object-cover" />
  <CardContent>
    <h3 className="font-semibold">产品名称</h3>
    <p className="text-muted-foreground">产品描述</p>
  </CardContent>
  <CardFooter>
    <span className="font-bold">¥99.00</span>
    <Button size="sm">购买</Button>
  </CardFooter>
</Card>
```

### 统计卡片

```tsx
<Card
  size="sm"
  title="总销售额"
  titleLeading={<TrendingUp className="text-success" />}
>
  <div className="text-2xl font-bold">¥12,345.67</div>
  <p className="text-muted-foreground text-xs">较上月 +12.5%</p>
</Card>
```

### 设置卡片

```tsx
<Card
  split
  title="通知设置"
  titleTrailing={<Badge variant="soft">3 项</Badge>}
  footer={(
    <div className="flex justify-end gap-2">
      <Button variant="ghost">取消</Button>
      <Button>保存</Button>
    </div>
  )}
>
  <div className="space-y-4">
    {/* 设置项 */}
  </div>
</Card>
```

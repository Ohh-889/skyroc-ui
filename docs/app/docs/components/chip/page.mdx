# Chip 徽标

徽标组件用于在元素上显示一个小圆点或数字，常用于提示未读消息数、通知数量等场景。

## 安装

Chip 组件是内置组件，无需额外安装依赖。

## 导入

```ts copy
import { Chip, ChipRoot, ChipContent } from 'skyroc-ui';
```

## 组合式 API

如果需要更灵活的布局控制，可以使用组合式 API：

```tsx
import { ChipRoot, ChipContent } from 'skyroc-ui';

export default function Demo() {
  return (
    <ChipRoot>
      <Button>消息</Button>
      <ChipContent color="destructive" position="top-right">
        99+
      </ChipContent>
    </ChipRoot>
  );
}
```

## API

### Chip Props

简化的徽标组件，封装了所有子组件。

| 属性 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| content | `ReactNode` | - | 徽标内容，不传则显示小圆点 |
| color | `'primary' \| 'destructive' \| 'success' \| 'warning' \| 'info' \| 'carbon' \| 'secondary' \| 'accent'` | `'primary'` | 徽标颜色 |
| position | `'top-right' \| 'top-left' \| 'bottom-right' \| 'bottom-left'` | `'top-right'` | 徽标位置 |
| size | `'xs' \| 'sm' \| 'md' \| 'lg' \| 'xl' \| '2xl'` | `'md'` | 徽标尺寸 |
| open | `boolean` | `true` | 是否显示徽标 |
| classNames | `ChipUi` | - | 自定义各部分的样式类名 |
| className | `ClassValue` | - | 根容器自定义类名 |

> 支持原生 `div` 元素的其他所有属性。

### ChipUi（classNames）

| 属性 | 类型 | 说明 |
|------|------|------|
| root | `ClassValue` | 根容器样式 |
| content | `ClassValue` | 徽标内容样式 |

### ChipRoot Props

徽标的根容器组件。

| 属性 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| className | `ClassValue` | - | 自定义类名 |

> 支持原生 `div` 元素的其他所有属性。

### ChipContent Props

徽标内容组件。

| 属性 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| color | `ThemeColor` | `'primary'` | 徽标颜色 |
| position | `ChipPosition` | `'top-right'` | 徽标位置 |
| size | `ThemeSize` | `'md'` | 徽标尺寸 |
| className | `ClassValue` | - | 自定义类名 |

> 支持原生 `span` 元素的其他所有属性。

## 位置说明

| 位置 | 说明 |
|------|------|
| `top-right` | 右上角（默认） |
| `top-left` | 左上角 |
| `bottom-right` | 右下角 |
| `bottom-left` | 左下角 |

## 使用指南

### 何时使用

- 消息通知数量提示
- 购物车商品数量
- 待办事项数量
- 新功能或更新提示
- 在线状态指示

### 颜色使用建议

- **Primary** - 一般通知
- **Destructive** - 重要通知、错误提示
- **Success** - 成功状态、在线状态
- **Warning** - 警告提示
- **Info** - 信息提示
- **Carbon** - 深色主题
- **Secondary** - 次要通知
- **Accent** - 强调提示

### 尺寸使用建议

- **xs/sm** - 小图标、紧凑型布局
- **md** - 默认尺寸，适合大多数按钮
- **lg/xl** - 大按钮、重要提示
- **2xl** - 特大按钮、营销场景

## 注意事项

1. **数字格式化** - 超过 99 时建议显示 "99+"
2. **隐藏控制** - 使用 `open={false}` 或条件渲染来隐藏徽标
3. **位置选择** - 根据内容布局选择合适的位置
4. **动态更新** - 徽标内容变化时会自动更新

## 无障碍

- 确保徽标内容对屏幕阅读器友好
- 使用适当的 `aria-label` 描述徽标含义
- 重要通知应有额外的文字说明

## 最佳实践

### 消息通知

```tsx
<Chip
  color="destructive"
  content={unreadCount > 99 ? '99+' : unreadCount}
  open={unreadCount > 0}
>
  <Button>消息</Button>
</Chip>
```

### 购物车

```tsx
<Chip
  color="primary"
  content={cartCount}
  open={cartCount > 0}
>
  <ButtonIcon>
    <ShoppingCart />
  </ButtonIcon>
</Chip>
```

### 在线状态

```tsx
<Chip
  color="success"
  position="bottom-right"
>
  <Avatar src={user.avatar} alt={user.name} />
</Chip>
```

### 动态显示

```tsx
'use client';

import { useState } from 'react';

export default function Demo() {
  const [count, setCount] = useState(5);

  return (
    <div className="flex gap-4">
      <Chip
        color="destructive"
        content={count}
        open={count > 0}
      >
        <Button>通知</Button>
      </Chip>
      
      <Button onClick={() => setCount(c => c + 1)}>+1</Button>
      <Button onClick={() => setCount(c => Math.max(0, c - 1))}>-1</Button>
    </div>
  );
}
```

### 多位置展示

```tsx
<div className="flex gap-8">
  <Chip position="top-left" color="primary">
    <div className="size-12 rounded bg-muted" />
  </Chip>
  <Chip position="top-right" color="success">
    <div className="size-12 rounded bg-muted" />
  </Chip>
  <Chip position="bottom-left" color="warning">
    <div className="size-12 rounded bg-muted" />
  </Chip>
  <Chip position="bottom-right" color="destructive">
    <div className="size-12 rounded bg-muted" />
  </Chip>
</div>
```
